# Satoshi Genesis Website Export

This is a ready-to-deploy Next.js website export for Satoshi Genesis.

## What's included
- One-page branded website
- Whitepaper PDF in `public/downloads/whitepaper.pdf`
- Pitch deck in `public/downloads/pitch-deck.pptx`
- Tokenomics graphic in `public/downloads/tokenomics-graphic.pdf`

## Run locally
1. Install Node.js 20+
2. In the project folder run:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`

## Deploy to Vercel
1. Create a new GitHub repo
2. Upload all project files
3. Push to GitHub
4. In Vercel, click **Add New Project**
5. Import the GitHub repo
6. Click **Deploy**

## Files you may want to replace later
- `public/downloads/whitepaper.pdf`
- `public/downloads/pitch-deck.pptx`
- `public/downloads/tokenomics-graphic.pdf`

If you create newer branded versions, replace these files with the same filenames and redeploy.
